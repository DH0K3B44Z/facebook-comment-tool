🔥 Facebook Auto Comment Tool Setup 🔥

📁 Required Files:

  • token.txt → Put your **access tokens**, one per line.
  • comment.txt → Put your **comments**, one per line.

📌 Input Format:

Just input the Post ID only. (No full link)

🛠️ Instructions:

  • Ensure both token.txt and comment.txt are uploaded in the **same directory** as the script.
  • Post ID should be copied directly from post URL, like: 123456789012345
  • You can type hater’s name or time manually wherever needed.

🚀 Example Token File:

EAABsbCS1i...long_token_1
EAAGm0PX4Z...long_token_2

💬 Example Comment File:

🔥🔥🔥 OP BRO
😂😂 TOO FUNNY
LEGEND MOVE 🤯

⚠️ Important:

  • Comment will be sent from tokens **one by one** (not simultaneously).
  • Same token will pause for 1 min to avoid block.
  • Tool will keep cycling tokens and comments.
